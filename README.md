# Практическая работа №1 по Django

Мини-приложение для управления пользователями, организациями и должностями.

## Функционал
- CRUD для организаций
- CRUD для должностей
- Связь пользователей с организациями на определённой должности
- Один пользователь может состоять в нескольких организациях
- В одной организации — только одна должность на пользователя
- Дефолтное значение в должности - "ассистент преподавателя"
